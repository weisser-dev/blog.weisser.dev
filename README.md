# blog.weisser.dev

Personal developer blog covering projects, AI tooling, infrastructure, and engineering notes.

Built with Jekyll and hosted on GitHub Pages.

## Structure

```
_posts/
├── development/   # General dev articles
├── linux/         # Linux & shell
├── microservices/ # Distributed systems, APIs
├── openclaw/      # OpenClaw AI agent series
├── operations/    # DevOps, deployment, infra
└── projects/      # Open-source projects & experiments
```

## Running Locally

```bash
gem install bundler
bundle install
bundle exec jekyll serve
# open http://localhost:4000
```
